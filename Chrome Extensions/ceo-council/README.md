# CEO Council — Chrome-расширение

Оценка сайта тремя «экспертами» (Стив Джобс, Марк Цукерберг, Артемий Лебедев). По клику на иконку — анализ текущей страницы через Claude; мнения выводятся на русском, по одному при каждом клике.

## Установка

1. **Бэкенд** (обязательно): в папке `backend` создай `.env` с `ANTHROPIC_API_KEY=sk-ant-...`, затем `node server.js`. Подробнее: [backend/README.md](backend/README.md).

2. **Расширение:** в Chrome открой `chrome://extensions/` → «Режим разработчика» → «Загрузить распакованное расширение» → выбери папку **`chrome-extension`** (внутри этой папки ceo-council).

3. Открой любой сайт, нажми на иконку расширения → «Анализировать сайт». Через 20–40 сек появятся три мнения; при следующих кликах они показываются по очереди.

## Структура

- `chrome-extension/` — файлы расширения (manifest, popup, content.js).
- `backend/` — Node-сервер на порту 3001, вызовы Anthropic API.
